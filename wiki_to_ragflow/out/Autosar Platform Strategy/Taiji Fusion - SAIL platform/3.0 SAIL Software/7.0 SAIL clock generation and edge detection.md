## Overview

This page is to analyze the solution for Clock generation and edge detection which is needed for USS in SAIL

QC MCAL package doesn't include the PWM and ICU module

## PWM generation

### Solution 1: Using Gpt and Dio (Workaround)

Can not go with this one, GPT timer is too slow

We can generate PWM using timer and GPIO module

- Configure the frequency of GPT timer
- Configure the interrupt for GPT timer
- In the ISR, we can toggle the Pin

Dependency:

- GPT and Port & Dio configuration

![](../../../_images/7.0%20SAIL%20clock%20generation%20and%20edge%20detection/image-2024-11-13_14-48-19.png)

### Solution 2: Using Clock IP

Better solution, we will go with this and make this concept work

QC has Clock generator hardware module which can be used to generate clock reference

Main domain also has similar clock hardware module

There is no MCAL configuration to configure clock, so we need to directly change the clock generator registers

Below is the register information to generate clock in SAIL_IO_59 pin

![](../../../_images/7.0%20SAIL%20clock%20generation%20and%20edge%20detection/image-2024-11-12_19-26-34.png)

We need to refer BSP code to detect GPIO edge detection and trigger interrupt
