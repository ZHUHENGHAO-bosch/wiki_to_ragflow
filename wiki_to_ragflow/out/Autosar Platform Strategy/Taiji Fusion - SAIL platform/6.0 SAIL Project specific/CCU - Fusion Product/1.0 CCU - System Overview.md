## Overview

CCU is a fusion product for SGM which will use TC387 as SCC and QAM8775 as SOC

Important part of this fusion is the complete use of SAIL domain for ADAS processing

### Block diagram

Overall Block diagram can be found in below path

16_CCU - All Documents (bosch.com)

Our focus is on Autosar domain in SCC and SAIL domain

| Domain | Peripheral | Description |
| --- | --- | --- |
| SAIL | CAN 5 CAN channels are connected to SAIL | PCAN - TJA1044 CCAN - TJA1044 Corner Radar - TJA1044 Front Radar - TJA1044 Reserved - TJA4011 |
| Ethernet - RGMII | Ethernet from Gateway ECU |
| SPI - 2 channels | USS is connected to SAIL via SPI |
| UART | Connected to SCC |
| OSPI | Connected to SNOR |
| SCC | SPI | Inertial sensor SMI860 |
| SPI (2 channels) | Connected to DSP NOR Flash |
| CAN (2 Channels) | ICAN - TJA1145 TCAN - TJA1042 |
| LIN (2 Channels) | SWC - Steering wheel switch SWT |
| GPIO | ICC key * 2 |
