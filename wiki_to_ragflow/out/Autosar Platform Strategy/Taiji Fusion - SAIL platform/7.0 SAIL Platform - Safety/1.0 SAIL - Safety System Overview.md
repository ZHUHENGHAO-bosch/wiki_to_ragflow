## Overview

QAM8755 Safety shall cover ADAS/ADs and IVI functions

ADAS and ADS use cases

- Highway autopilot (HWP)
- Traffic jam assist
- Automatic emergency braking and lane keep assist
- Automated valet parking

IVI use cases

- Instrument cluster tell-tale icon display
- Surround view systems with object detection and free space estimation
- Rear view camera
- Driver monitoring system/occupant monitoring system
- e-Mirror

Safe computing sub systems in QAM8775

- Application processor subsystem (APSS)
- Graphics processing unit (GPU)
- Safety island (SAIL)
- Neural signal processor (NSP)
- Computer vision processor (CVP)

Our focus here is on SAIL so we move towards SAIL safety

SAIL domain can achieve ASIL-D level whereas APSS can achieve up to ASIL-B
