### Localization of CSB

Currently the CSB (Country Specific board) is using SBR ( Software-Defined Radio )

What is SBR ? Software-Defined Radio (SDR) is a radio communication system where components traditionally implemented in hardware (like mixers, filters, modulators/demodulators) are instead processed using software on a computer or embedded system. This allows greater flexibility, reconfigurability, and adaptability compared to conventional radios

Motivation: Global Civic based on CSB used SDR using Renesas RCar3 which is high cost, for CN market to reduce the cost, MB wants to replace SDR with HW defined radio with single core MCU.
