## Manifest of Project for Config modules

Options:

1. It will be good if we have config repos for each project
2. Another approach is to create branch for each project
3. Another option is to have different project folders inside the config folders -> But when we download one project repo, then it will have project folders inside config folders which is not good

Decision:

We will create one config repos for project and for next project we will create branch out of each config repos
