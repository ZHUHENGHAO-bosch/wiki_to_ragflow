For Aurix China convergence platform and projects, We have decided to use UTF (Unified Test Framework) and RFW (Robotic Framework) for integration testing.

We have predefined RFW scripts (which uses DLT plugins) to write our test cases which will execute in Host system.

UTM (UTF module in SCC) will be running in SCC side and will receive message via DLT and execute test cases based on input command from RFW

Advantage of UTF over XTM

- Test Case logic is completely separated from the actual business logic
- Separate binary for each Test Module
- Usage of Wrap functionality - 6. Creating Wrap functions - XC-CT China - Docupedia (bosch.com)
