### Steps to add Test cases inside an module

Step 1: Write the test case in the respective modulename_TestCase.h (in our example, LCM_TestCase.h) file add your test case function in order. This file will be scanned by UTF generator tool to create test list in generated Modulename_TCContainer.c (in our case it is LCM_TCContainer.c) file

![](../../../../_images/3.%20Adding%20Test%20Case%20for%20Module%20-%20UTF/image-2023-3-25_16-1-55.png)

Define the test case function in modulename_TestCase.c (in our example, LCM_TestCase.c)

Make sure the test case in modulename_TestCase.h is reflected in Modulename_TCContainer.c (in gen folder Config\Autosar\integration-test\Service.UTFManager.Aurix\gen)
