### Below is the command format for UTF

| Byte Order | Description | Comment |
| --- | --- | --- |
| First byte | Type of Testing | 00 UTF Module Testing \| 01 UTF Module Test Case |
| Second byte | Input Length | Total command length |
| Third and fourth byte | Module ID | Module ID is defined in UTFModuleDetails_t in the file "gen\UTFManager_cfg.h" |
| Fifth and sixth byte | Test Case ID | The order which we define in Modulename_TestCase.h |
| Seventh byte and .... | Payload data to test case | Data that can be used for each test case |

Example,

To trigger a test case 1 in Module 1

01 06 00 01 00 01 00 02 XX XX
