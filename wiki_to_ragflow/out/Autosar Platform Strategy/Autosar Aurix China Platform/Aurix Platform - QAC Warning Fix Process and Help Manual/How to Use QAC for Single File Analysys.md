## Step1: Build your software.

## Step2: Generate QAC project.

Move to path: \Config\Autosar\Build\HelixQAC\Tool.PRQA.

Run command: .\GenerateQAC.bat --CX1E

QAC project will be generated in path: \Config\Autosar\Build\PRQA_CX1E

## Step3: Open QAC

Move to path: \Config\Autosar\Build\PRQA_CX1E

double click on: OpenProjectInGUI.bat

## Step4: Choose file to Analysis

Select file or folder;

Right click and choose Analyze Selected File(s).

![](../../../_images/How%20to%20Use%20QAC%20for%20Single%20File%20Analysys/image-2023-8-9_15-49-25.png)

## Step5: Check the Warnings and Fix

Warnings will be shown in the bottom of the QAC tool.

Click on the ID and check the help manual to fix the issue.

![](../../../_images/How%20to%20Use%20QAC%20for%20Single%20File%20Analysys/image-2023-8-9_15-56-18.png)
