## General

This page is to summarize

### Use case

| Use case (voice / AI agent) | Hazard if LLM gets it wrong | Required mediation |
| --- | --- | --- |
| S1 | "Open all doors" / unlock | Door opens at speed → occupant ejection, child fall-out | Block if speed > 5 km/h; child-lock state respected |
| S2 | "Open the trunk" while driving | Trunk-open at speed → crash, debris, visibility | Block if speed > 0; key authentication |
| S3 | "Roll up window" with limb/object detected | Pinch injury, especially child | Anti-pinch sensor must override; voice never bypasses |
| S4 | "Open the sunroof" in rain / car-wash / high speed | Water ingress, wind injury, structural noise distraction | ODD checks: rain sensor, speed, wash mode |
| S5 | "Apply / release parking brake" (EPB) — see prior turn | Loss of standstill on slope, dynamic brake at speed | Hard allow-list: voice EPB only at v=0, gear P/N, confirmed |
| S6 | "Turn off headlights" / "Turn off hazards" while driving at night / emergency | Loss of conspicuity, rear-end collision | Block; reverse the request — explain rather than execute |
| S7 | "Disable Lane-Keep / AEB / ACC" | Removes driver-assistance safety net silently | Two-stage confirmation; cluster banner; ASIL boundary respected |
| S8 | "Change drive mode to Sport / Off-Road / Snow" | Wrong traction model on current surface → loss of control | Context-aware: speed, gear, surface estimate; confirm |
| S9 | "Maximize regenerative braking" on low-µ surface | Rear lock-up, slide | ESC-arbitrated; refuse if µ-estimate low |
| S10 | "Auto-park" / "Summon" / "Drive me home" | Beyond cockpit ODD — LLM must not initiate AD | Hard block; route only via ADAS HMI which has its own ASIL D |
| S11 | "Lock the doors now" during a crash / emergency egress | Trapped occupants | Crash signal & door-handle override always win |
| S12 | Charging / high-voltage commands ("Start fast-charge") | HV exposure during service, thermal event | Lockout in Service Mode, plug-state checks |
| S13 | "Fold the mirrors" / "retract" while driving | Visibility loss | Speed gate |
| S14 | "Recline driver seat" / move seat while driving | Driver loses pedal reach, airbag pose violated | Block on driver seat above 0 km/h; passenger seat with limits |
| S15 | Child-lock toggle by voice | Child opens door in traffic | Driver-only voice biometric + confirmation |
| S16 | Tow-mode / Transport-mode toggle | Brake/Park-lock behavior changes unexpectedly | Service-mode authentication |
| S17 | Send eCall / cancel eCall by LLM | False emergency calls or suppressed real ones | LLM may suggest ; cannot initiate or cancel eCall |
