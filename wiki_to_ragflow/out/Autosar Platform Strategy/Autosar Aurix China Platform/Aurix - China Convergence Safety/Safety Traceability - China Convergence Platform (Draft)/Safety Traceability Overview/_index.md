### Decision:

To support the project timeline and requirements, we will go with Solution 1. Once we make the exchange and linking properly, it will be easy for the future China projects

## Solution1: With both CART and China project in DOORS classic (DOORS exchange):

Pros:

- Control is now completely with project

Cons:

- Complex structure
- Needs DOORs exchange and need to run script
- Need inspection review to make sure links are established properly

![](../../../../../_images/Safety%20Traceability%20Overview/CnConvPf_Traceability.png)

## Solution 2: With CART in DNG and China project in DOORS classic (using specified by):

Pros:

- Simple and less work
- Project can concentrate only on Config part.
- No big linking and inspection review effort for PF framework

Cons:

- PF part control is completely with CART
