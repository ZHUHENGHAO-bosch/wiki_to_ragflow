## Overview

This page is to analyze the Chime manager application module

![](../../../../_images/3.0%20Chime%20Manager/1111.png)

### Chime Manager deployment

Chime Manager decides which chime to play based on

- CAN messages
- Action messages from HMI application
- Chime Producer status and
- Priority of Chime event

Chime manager after deciding which chime to play, will send request to Chime producer in SoC side.

Chime producer will load the audio file based on Chime id and request Audio diver to play the Chime
