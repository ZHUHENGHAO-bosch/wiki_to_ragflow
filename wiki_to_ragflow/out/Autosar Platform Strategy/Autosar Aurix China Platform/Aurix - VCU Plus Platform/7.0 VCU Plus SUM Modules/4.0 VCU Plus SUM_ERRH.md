## Overview

The Standard Utility Module Error Handler (SUM_ERRH) is a software composition located in the Application Layer of the standardized architecture of AUTOSAR.

This error may be responsible by Diagnostic team

### Main Tasks of SUM_ERRH

- SUM_ERRH is notified of error events created by BSW Production Errors or SUM error events
- SUM_ERRH filters the events with the appropriate debounce algorithm. Based on the result of the debounce algorithm, SUM_ERRH notifies Dem of an event and a test pass/fail for that event. Dem is responsible to maintain the status byte
- Aggregation of BSW production error events from Dem
- Aggregation of SUM error events from each SUM
- Aggregation of deadline monitoring events from SUM_SSM for signals which are relevant to setting of node loss U-code DTC
- Aggregation of DTC list from Dem
- Precondition Check for Control DTC Setting request from DCM based on Channel and Tester information
- Precondition Check for Clear Diagnostic request from DCM for OBD/Non-OBD ECUs
