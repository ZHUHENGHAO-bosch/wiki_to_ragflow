## Overview

The Standard Utility Module, Global Time Client Service (SUM_TCS), is a software composition located in the Application Layer of the standardized architecture of AUTOSAR.

SUM_TCS is primarily intended to be used to provide a synchronized time stamp for the extended date record of a DTC

SUM_TCS receives a signal (TimeSyncMessage) via the serial data communication bus from an ECU which has the MasterClock function. The resolution and transmission rate of the Master Clock signal is as defined by the diagnostic team

### Functionalities of SUM_TCS

- Receive the global time from the Master Clock global time provider via serial data communication Synchronize the local ECU time to the master global time
- Substitute the last received value (default if never received) following initialization, until the initial Master Clock signal is received via serial communication
- Keep a local timer source to maintain ECU internal time source between reception events of the Master Clock signal.
- Provide a DID Interface to the diagnostic module DCM, providing current timing client local time value
