## Overview

This page is to analyze the MCU chip which is MEC172x from Microchip

### Features

MEC172x = Laptop / PC Embedded Controller

It sits between the CPU and hardware devices and manages power, keyboard, thermal, and system control tasks .

Features of MEC172x

- ARM Cortex-M4F Embedded Processor
- 96 MHz Internal PLL
- Package
- Boot ROM Secure Boot Loader
- Memory
- Peripherals
